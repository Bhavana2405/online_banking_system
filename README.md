# Online Banking System

## Description
A simple Online Banking System developed using Python and Tkinter. This project provides a graphical user interface for basic banking operations.

## Features
- User Login
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python 3.
2. Download the project.
3. Open the project folder.
4. Run:
   ```
   python banking_system.py
   ```
